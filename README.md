# DigiTools - Digital Tools Buying Website

## Project Description

DigiTools is a responsive React-based digital tools buying website where users can explore premium products, add them to cart, remove selected items, and complete checkout with interactive toast alerts.

## Technologies Used

- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify
- JSON data

## Core Features

1. Product and Cart toggle section with default product view.
2. Full cart workflow: add product, remove product, view total, proceed checkout.
3. Responsive UI with meaningful content and Figma-inspired design.

## Optional Features Implemented

- Navbar cart count increases with add-to-cart and resets after checkout.
- Buy Now button text temporarily changes to “Added to cart”.
